---
title: Reverify Password
description: Request the user to re-verify their identity by entering their password for revalidation.
---

Sometimes you may want to re-verify the user's identity by entering their password. This can be useful for scenarios such as:

- When the user logs in after a long time
- When the user requests a critical action (e.g. deleting their account)

<Steps>
  <Step>
        ### 1. Install the plugin

        ```package-install
        @better-auth-kit/reverify-password
        ```
  </Step>

  <Step>
    ### 2. Initialize the plugin

    ```ts title="auth.ts"
    import { reverifyPassword } from "@better-auth-kit/reverify-password";

    export const auth = betterAuth({
      plugins: [reverifyPassword()],
    });
    ```
  </Step>
  <Step>
    ### 3. Use the plugin

    <Tabs items={["client", "server"]}>
      <Tab value="client">
        ```ts
        const isValid = await authClient.reverifyPassword("some-password");
        ```
      </Tab>
      <Tab value="server">
        ```ts
        const isValid = await auth.api.reverifyPassword("some-password");
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
