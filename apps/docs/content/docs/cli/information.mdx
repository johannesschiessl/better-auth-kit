The intention of the Better Auth Kit CLI is to provide a set of commands that help you with your Better Auth project.

<Callout type="warn">
Currently, the CLI is in the early stages of development and not all commands are available.
Check out our [roadmap](/roadmap) to see what is coming soon.
</Callout>

## Features

- [UI Components](/docs/cli/ui-components) - Generate UI components for your project.
- [Seeding](/docs/cli/seeding) - Create fake users or other data in the database that relate to Better-Auth.
- [Better Auth Dashboard](/docs/cli/dashboard) - A Clerk-like dashboard for your better-auth project.
